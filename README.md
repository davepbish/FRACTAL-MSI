# FRACTAL-MSI
Code and data used to create images in FRACTAL-MSI

Frac_MSI_IF_register.py is the code to register immunofluorescent and LA-ICP-MS images.
frac_image_process.py is the code to do image fusion by the CDFIF and PRIF methods of the registered multimodal images.
Example images used to make figures in the manuscript can be found in the 'rawfiles' folder.
